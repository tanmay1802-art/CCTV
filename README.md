# 🛡️ Python-based CCTV & Screen Recorder

Python-based CCTV and screen recording system using OpenCV and PyAutoGUI for monitoring.

This is a lightweight desktop monitoring utility built with **OpenCV** and **PyAutoGUI**.
It records real-time screen activity and saves it as an AVI video file.

## ✨ Features
- Real-time screen recording
- Automatic video saving
- Designed for monitoring and auditing purposes

## 📦 Requirements
- Python 3.8+
- opencv-python
- pyautogui
- numpy
- pillow

pip install -r requirements.txt
