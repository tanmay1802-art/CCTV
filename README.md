# Python-based CCTV and Screen Recorder

This project is a lightweight screen monitoring and recording system developed using Python. It uses OpenCV and PyAutoGUI to capture real-time screen activity and save it as a video file. The project is designed to simulate a basic CCTV-style monitoring system for educational and demonstration purposes.

---

## Features

* Real-time screen recording
* Automatic video saving in AVI format
* Simple and lightweight implementation
* Suitable for monitoring and auditing demonstrations

* opencv-python

* pillow
